# IONET

INSTALLATION
```
git clone https://github.com/ffoster007/IONET.git

chmod +x ionet.sh

./ionet.sh install

<---Now it can ready to use--->

## UNINSTALLATION
*  ionet uninstall

```

## USAGE
```
ionet [options] <ip_address or domain>
```
